# Step 2: Configuration

Set up your project configuration and environment variables.

## Configuration Steps

1. Create a `.env` file
2. Add your API keys
3. Configure your database connection


{
  "apiKey": "your-api-key",
  "dbUrl": "mongodb://localhost:27017",
  "port": 3000
}


{% hint style="warning" %}
Never commit sensitive credentials to version control!
{% endhint %}
